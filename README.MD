# Student Task Manager

🎯 A fun, student-first task manager built with plain HTML, CSS, and JavaScript.

This static web app helps you track:
- assignments and due dates
- course progress
- exam schedules
- resource downloads

> Open it locally in your browser. No server needed.

## 🚀 What makes this project interesting

- Clean dashboard for homework and course planning
- Interactive calendar with event indicators
- Exam schedule and course resource sections
- Mobile-friendly layout with modern card styling
- Simple demo login flow for a classroom prototype

## 📁 Key pages

- JavaScript

## 💡 How to use

1. Open `dashboard.html` in a browser, or launch with Live Server in VS Code.
2. Navigate between pages using the top menu.
3. Click on calendar dates and explore exam/resource cards.
4. Use the demo login fields if the app includes a login page.
